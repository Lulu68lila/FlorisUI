FlorisUI Docs v1.0.1

Apprenez à installer, utiliser et personnaliser FlorisUI dans vos projets.

Installation

Via npm (recommandé)

npm install florisui

Via CDN

<link rel="stylesheet" href="https://unpkg.com/florisui/dist/florisui.min.css">
<script src="https://unpkg.com/florisui/dist/florisui.min.js"></script>

Via fichier

Téléchargez les fichiers depuis le dépôt GitHub et liez-les dans votre page :

<link rel="stylesheet" href="css/style.css">
<script src="js/main.js"></script>

Usage

Ajoutez la classe bloom à n'importe quel conteneur pour activer le thème FlorisUI :

<div class="bloom">
  <button class="btn btn--primary">Click me</button>
</div>

Le CSS utilise des variables CSS personnalisables. Vous pouvez les surcharger dans votre propre feuille :

:root {
  --violet: #8b5cf6;
  --pink: #f472b6;
  --cyan: #22d3ee;
  --bg-primary: #0a0820;
  --text-primary: #f1e8ff;
}

Personnalisation

Toutes les couleurs et dimensions sont contrôlées via des variables CSS. Surchargez-les pour créer votre propre thème :

.my-theme {
  --violet: #6c3bd6;
  --pink: #e85d9a;
  --cyan: #1cc4d8;
  --radius-lg: 20px;
  --glass-bg: rgba(108, 59, 214, 0.08);
}

const input = document.querySelector('.bloom-slider__input');
input.addEventListener('input', () => {
  document.querySelector('.bloom-slider__fill').style.width = input.value + '%';
  document.querySelector('.bloom-slider__thumb').style.left = input.value + '%';

React — Installation

florisui-react fournit des composants React prêts à l'emploi avec le style FlorisUI intégré.

npm install florisui-react

Importez le CSS et les composants dans votre projet :

import 'florisui-react/dist/style.css';
import { Button } from 'florisui-react';

function App() {
  return <Button variant="primary">Hello FlorisUI</Button>;
}

Button (React)

Bouton avec effet de glow qui suit la souris.

import { Button } from 'florisui-react';

<Button variant="primary">Primary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="primary" glow={false}>Sans glow</Button>
PropTypeDéfautDescription
variant'primary' | 'ghost''primary'Style du bouton
glowbooleantrueAffiche l'effet de lumière
childrenReactNode—Contenu du bouton

Toggle (React)

Interrupteur organique avec mode contrôlé ou non-contrôlé.

import { Toggle } from 'florisui-react';

// Non-contrôlé
<Toggle label="Activer" />

// Contrôlé
<Toggle checked={isOn} onChange={setIsOn} label="WiFi" />
PropTypeDéfautDescription
checkedboolean—État contrôlé
onChange(checked: boolean) => void—Callback de changement
labelstring—Texte affiché à côté

Slider (React)

Slider avec remplissage dégradé et valeur affichée.

<Slider min={0} max={100} defaultValue={50} />

Accordion (React)

Accordéon avec pétale animé. Fonctionne avec une liste d'éléments.

const items = [
  { id: '1', icon: '🌸', title: 'Titre', content: 'Contenu...' },
  { id: '2', icon: '🎨', title: 'Titre 2', content: 'Contenu...' },
];

<Accordion items={items} />
PropTypeDescription
itemsAccordionItem[]Liste des éléments

AccordionItem : {id: string, icon: string, title: string, content: string}
